This repository contains two Python tasks to understand **dictionaries**, **user input handling**, and **list slicing operations**.

## Task 1: Create a Dictionary of Student Marks

### Problem Statement
Write a Python program that:
1. Creates a dictionary where **student names are keys** and **their marks are values**.
2. Asks the user to input a **student's name**.
3. Retrieves and displays the **corresponding marks**.
4. Displays an appropriate message if the **student name is not found** in the dictionary.



### Objective
- Understand how dictionaries work in Python
- Learn how to access dictionary values using keys
- Handle cases where a key may not exist



### Concepts Used
- Python dictionaries
- User input using `input()`
- Conditional statements (`if-else`)
- Dictionary key lookup




## Task 2: Demonstrate List Slicing

### Problem Statement
Write a Python program that:
1. Creates a list of numbers from **1 to 10**
2. Extracts the **first five elements** from the list
3. Reverses the extracted elements
4. Prints both the **extracted list** and the **reversed list**



### Objective
- Learn how list slicing works
- Understand reversing lists using slicing
- Practice list operations



### Concepts Used
- Lists in Python
- List slicing (`[:]`)
- Reverse slicing (`[::-1]`)
- Printing formatted output
